# ThreatWeave

Threat intelligence knowledge graph with deterministic correlation and reserved
AI enrichment. Work in progress — see the roadmap below.

## Roadmap

- [ ] **Phase 1 — Base graph**: project skeleton, deterministic IOC parsing, OTX
  ingestion, Neo4j graph model, correlation and a FastAPI query endpoint. No AI.
- [ ] Phase 2 — AI extraction of IOCs/TTPs from free text.
- [ ] Phase 3 — Embeddings + pgvector semantic correlation.
- [ ] Phase 4 — On-demand explanatory narratives.
